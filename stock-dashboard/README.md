# Personal Stock Board

개인용 주식 대시보드입니다. 관심종목, 보유수량, 평균단가를 한 화면에서 보고 자동 갱신합니다.

## 실행

브라우저에서 `index.html`을 열거나, 폴더에서 정적 서버를 실행합니다.

```powershell
python -m http.server 5173
```

그 다음 `http://localhost:5173`으로 접속합니다.

## 실시간 시세

- API 토큰이 없으면 데모 가격이 실시간처럼 움직입니다.
- 실제 시세는 화면의 `Finnhub API Token`에 토큰을 입력하고 저장하면 가져옵니다.
- 무료 API는 종목, 거래소, 요금제에 따라 지연 시세이거나 제한이 있을 수 있습니다.

## 심볼 예시

- 미국: `AAPL`, `NVDA`, `TSLA`, `MSFT`
- 한국: `005930.KS`, `000660.KS`

투자 판단은 본인 책임이며, 이 앱은 개인 모니터링 도구입니다.
