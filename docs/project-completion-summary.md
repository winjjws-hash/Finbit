# Energy Budget 프로젝트 정리

## 프로젝트 소개

Energy Budget은 오늘의 피로도, 기분, 남은 시간을 바탕으로 하루 에너지 예산을 계산하고 감당 가능한 일을 추천하는 모바일 앱이다.

## 완료한 산출물

| 구분 | 완료 내용 |
|---|---|
| 기획 | 비전, 문제 정의, 핵심 가치, 목표 사용자 |
| 요구사항 | 핵심 사용자 시나리오 3개, MoSCoW 기능 분류 |
| 일정 | 10주차부터 15주차까지 WBS와 Gantt 정리 |
| 설계 | ADR, Mermaid 아키텍처, 폴더 구조 설명 |
| 개발 환경 | Expo, React Native, TypeScript 기반 프로젝트 구성 |
| 발표 | GitHub Pages 발표 화면, 2분 발표 대사, PDF 자료 |

## 현재 상태

기획과 설계 단계는 발표 가능한 수준으로 정리되었다. 개발은 Expo 기반 구조와 첫 화면 흐름을 준비한 상태이며 다음 단계에서 컨디션 입력과 할 일 등록 기능을 연결한다.

## 다음 구현 목표

1. 컨디션 입력 화면 완성
2. 할 일 등록 화면 완성
3. 에너지 예산 계산 규칙 연결
4. 오늘 추천 목록과 미루기 후보 표시
5. 일일 계획 저장과 주간 리포트 구성
