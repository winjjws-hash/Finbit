# Energy Budget 11주차 목표 완료 정리

## ADR 설명

React Native, Expo, TypeScript를 선택했다. 혼자 6주 안에 모바일 MVP를 만들기 쉽고, Expo를 사용하면 빠르게 실행과 확인이 가능하기 때문이다.

## 구조 설명

새 화면은 `src/presentation/screens`에 둔다. 화면 데이터와 기능 흐름은 `src/application`에 둔다. 에너지 계산 규칙은 `src/domain/rules`에 둔다. 저장 기능은 `src/data`에서 담당한다.

## 실행과 인수인계

프로젝트 실행 방법, 아키텍처 설명, WBS, 발표 자료를 GitHub Pages와 문서로 정리했다. 발표자는 URL을 열고 2분 안에 프로젝트의 비전, WBS, 설계 구조, 다음 계획을 설명할 수 있다.
