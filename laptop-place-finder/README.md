# Laptop Place Finder

노트북 작업하기 좋은 장소를 지도와 추천 목록으로 찾는 정적 웹 앱 프로토타입입니다.

## 실행

`index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

## 기능

- 위치 검색 및 현재 위치 기반 추천
- Nominatim으로 검색 위치 좌표 확인
- Overpass API로 주변 실제 카페, 도서관, 공유오피스 조회
- 지도 핀 클릭으로 주변 장소 확인
- 콘센트, 테이블, 와이파이, 조용함 점수 기반 추천
- 장소 상세, 리뷰 목록, 리뷰 작성
- 마이페이지 탭에서 저장 장소와 내가 쓴 리뷰 확인
