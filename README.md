# Energy Budget

Energy Budget은 사람들의 하루 계획을 시간 중심에서 에너지 중심으로 바꾸어, 무리하지 않아도 꾸준히 성장할 수 있는 지속 가능한 일상을 만드는 모바일 앱입니다.

## Week 11 목표

- React Native + Expo + TypeScript 기반 초기 앱 구조
- Presentation / Application / Domain / Data 레이어 구조
- Hello World 수준의 첫 실행 화면
- `docs/setup.md`, `docs/architecture.md`
- ADR-0001 ~ ADR-0003 결정 기록

## 빠른 실행

```bash
npm install
npm run start
```

Expo 실행 화면이 열리면 Android/iOS Expo Go 앱 또는 웹 미리보기로 확인할 수 있습니다.
