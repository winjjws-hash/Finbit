# Energy Budget - 중간 발표 초안

## 1. 발표 제목

Energy Budget: 시간 중심 계획에서 에너지 중심 계획으로

## 2. 한 줄 소개

Energy Budget은 사람들의 하루 계획을 시간 중심에서 에너지 중심으로 바꾸어, 무리하지 않아도 꾸준히 성장할 수 있는 지속 가능한 일상을 만드는 모바일 앱이다.

## 3. 문제 정의

기존 투두앱은 해야 할 일을 많이 등록하고 완료했는지 확인하는 데 집중한다. 하지만 실제 생활에서는 시간이 있어도 피곤하거나 기분이 좋지 않아서 계획을 지키지 못하는 경우가 많다.

Energy Budget은 이 문제를 "시간 부족"이 아니라 "에너지 관리 부족"으로 본다.

## 4. 핵심 사용자 시나리오

1. 아침에 피로도, 기분, 남은 시간을 입력하고 오늘 감당 가능한 계획을 세운다.
2. 하루 중 계획이 무너졌을 때 남은 에너지 기준으로 다시 조정한다.
3. 일주일 기록을 보고 어떤 날에 과부하가 많았는지 확인한다.

## 5. Must 기능

- 오늘의 컨디션 입력
- 할 일 등록
- 에너지 예산 계산
- 오늘 할 일 추천
- 일일 계획 결과 저장

## 6. 기술 선택

| 결정 | 선택 |
|---|---|
| 모바일 플랫폼 | React Native + Expo + TypeScript |
| 상태관리 | React 기본 상태 + ViewModel 함수 구조 |
| 백엔드 | 초기 MVP는 로컬 저장소 우선 |

## 7. 아키텍처

Energy Budget은 다음 네 개 레이어로 구성된다.

- Presentation: 화면
- Application: 화면에 필요한 데이터 흐름
- Domain: 에너지 계산 규칙
- Data: 저장 기능

## 8. 현재 구현 상태

- Expo 프로젝트 설정 완료
- Hello World 첫 화면 구현
- Energy Budget 샘플 에너지 예산 표시
- 레이어드 디렉토리 구조 생성
- `docs/setup.md`, `docs/architecture.md` 작성
- ADR 문서 작성

## 9. 다음 구현 계획

1. 컨디션 입력 화면 구현
2. 할 일 등록 화면 구현
3. 에너지 예산 계산 결과를 실제 입력값과 연결
4. 추천 결과 화면 구현
5. 일일 기록 저장 구현

## 10. 발표 마무리 문장

Energy Budget은 더 많은 일을 하도록 압박하는 앱이 아니라, 오늘의 나에게 맞는 계획을 세우도록 돕는 앱이다. 이 프로젝트를 통해 사용자의 컨디션을 고려한 계획 추천 흐름을 모바일 앱으로 구현하고자 한다.
